# Programas Info — Agora Bot 2

## Objetivo

O Agora Bot 2 é a base oficial e independente para atendimento WhatsApp multiempresa.

## Escopo da primeira versão

- Backend standalone.
- MongoDB Atlas no banco `Agorabot2`.
- Meta WhatsApp Cloud API.
- Z-API como provider alternativo.
- Groq como IA oficial.
- Catálogo interno em MongoDB.
- Atendimento humano via `human_queue`.
- Socket.IO para eventos em tempo real.

## Fora do escopo desta etapa

- Integração com Agora Cortex.
- Painel frontend completo.
- Migração automática do banco legado `whatsapp`.
